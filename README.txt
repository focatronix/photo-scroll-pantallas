Galeria Tipo Masonry con Scroll Automatico Vertical
utilizando sistema de archivos 

Configurado en 4 columnas. 
Para mas columnas modificar las variables del scss, compilar 
un nuevo css y actualizar los datos de masonryLayout en index.js


Instrucciones:

1) Cargar las imagenes a presentar en la carpeta "Images" 

2) Abrir Index.html en Google Chrome

3) Oprimir "Choose Files" y seleccionar las imagenes.

4) Oprimir "Cargar Fotos" y esperar que se genere la galeria

5) Darle Play con el boton "Auto Scroll"

6) Al final se puede resetear.

Troubleshooting: Recargar index.html haciendo CONTROL + F5
